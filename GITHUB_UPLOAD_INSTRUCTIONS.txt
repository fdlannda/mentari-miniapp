# INSTRUKSI UPLOAD KE GITHUB
# 
# Jalankan command berikut di PowerShell (ganti YOUR-USERNAME dengan username GitHub Anda):
#
# git branch -M main
# git remote add origin https://github.com/YOUR-USERNAME/mentari-miniapp.git
# git push -u origin main

# git remote add origin https://github.com/john123/mentari-miniapp.git
#
# Setelah menjalankan, Anda akan diminta login GitHub (jika belum)
# Masukkan username dan password/token GitHub Anda
